### Git初始化

#### 4.1 创建版本库及第一次提交

查看git版本：`git --version`

在开始使用git之前，需要配置一下git的变量

    1. 告诉git当前用户的姓名和邮件
        git config --global user.name 'Your Name'
        git config --global user.email 'You Email'
    
    2. 设置一些git的别名
        git config --system alias.st status
        git config --system alias.ci commit
        
    3. 在git命令输出中开启颜色显示
        git config --global color.ui true
